## Pacotes instalados

```shell
npm init -y

npm i express -s

npm i cors

npm i https -s

npm i fs -s

npm i mysql2 -s

npm i body-parser -s

npm i nodemon -s-dev

use nodemon index.js para rodar a aplicação 

use npm ls para ver as dependências instaladas 
na aplicação ou veja isto no package.json
```
